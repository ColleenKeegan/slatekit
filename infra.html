Infrastructure

Save time and costs using pre-built architecture.
Focus on your business features.

Click on any item below to see more details and docs.

The architectural components such as queues, file storage, data access are built for AWS, Linux, MySQL


APIs

A simplified interface and abstraction for any type of persistent queue. We offer built in support for AWS SQS as well as a simple in-memory queue.

App

A foundation for any command line, batch, or other app with support for logging, configuration, environment selection, diagnostics, status and much more.

CLI

Command line access to your services, apis, and/or admin features. Extend the Shell app, register apis, and have the shell marshal commands via reflection.

Data / ORM

A convenient set of tools for accessing, querying data, and executing commands on your database via JDBC. Also has support for mapping objects.

Files

A simplified interface and abstraction for any type of persistent file storage. We offer built in support for AWS S3 to store files

Notifications

Background tasks/worker/jobs pattern to handle 1 time, long-running or continuous background processing. Built in support for queue processing available.


Queues

A simplified interface and abstraction for any type of persistent queue. We offer built in support for AWS SQS as well as a simple in-memory queue.

Server

A simplified interface and abstraction for any type of persistent file storage. We offer built in support for AWS S3 to store files

Tasks

Background tasks/worker/jobs pattern to handle 1 time, long-running or continuous background processing. Built in support for queue processing available.
