Slate Kit

Overview

A simple toolkit, utility library and server backend in Scala.

About

Slate Kit is a utility library, tool kit, and server backend for mobile and web apps. It is written in Scala and uses Akka-Http for the Server. While there are a few server backend projects available either for purchase or open-source, Slate Kit is one of the few written entirely in Scala, Akka-Http and integrated with AWS. Some of the other core differences with Slate Kit is the focus on simplicity, utility code and modularity above all else. One of the other core design philosophies has been to build protocol independent API's that can be "embedded" into a Web API Server or "embedded" into a CLI (Command Line Interface) Shell. Slate Kit is currently used to build the backend architecture for 2 mobile apps currently in production for 2 client projects of Code Helix Solutions Inc.


Philosophy

Simplicity above all else


Tech 100% Scala for all utilities, features, server code. Built for MySql, AWS ( S3, SQS )
Tool Kit This is NOT a framework. This is designed as a Tool Kit. Use as little or as much as you need
Simplicity Simplicity above all else
Responsibility All utilities, features, and architecture components designed for a single purpose.
Dependencies Dependencies on external, 3rd-party libraries are kept to a minimum
Conventions Following the idea of convention over configuration
Balance
  • Immutable Data + minimal areas of mutability
  • Idiomatic Scala + areas of divergence for simplicity
  • Strongly Typed + areas of dynamic objects/reflection calls
