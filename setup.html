Setup

Setup SlateKit for rapid development.

SlateKit will soon be available on package management tools like sbt, maven, gradle


Requirements

Java 1.8 For all code
Scala 2.17 For all code


Optional

Sbt latest Build tool
MySql Connector 5.7 For JDBC connections
Akka-Http latest For Slate Server
AWS Sdk ( For Java ) latest Cloud Storage of Files(S3), Queues(SQS)
IntelliJ latest IDE


Installation

Java

  • install java 1.8.0 to example: C:\Tools\Java\jdk1.8.0
  • setup java home env variable: JAVA_HOME=C:\Tools\Java\jdk1.8.0

Scala

  • install scala from download url below to c:\tools\scala\lang
  • ensure C:\tools\scala\lang\bin; is in your system PATH variable.
  • ensure environment variable SCALA_HOME=C:\Tools\Scala\Lang

SBT

  • install sbt ( scala build tool ) from download url below to c:\tools\scala\sbt
  • ensure C:\tools\scala\sbt\bin; is in your system PATH variable.
  • ensure environment variable SBT_HOME=C:\tools\Scala\SBT\

Slate

  • download binaries or source code


Configuration

There is currently no specific configuration needed at a global level for Slate Kit.

Compiling

Slate

  • download binaries or source code
  • if you are using binaries, drop jar files into your sbt lib folder
  • if you are using sources, add project dependencies to the slate kit projects.
