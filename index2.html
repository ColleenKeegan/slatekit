Slate Kit

Slate Kit is a Kotlin based open-source collection of libraries. It is comprised of several modular components and pre-built architecture that can be used to build any type of applications on the JVM. It can also be used as an open-source alternative to FireBase/Parse Beta* as the server-side backend for your applications.

Utilities

A large suite of common utilties to supplement the standard library

Architecture

Pre-built architecture components for both client and server

Cloud

Integration with AWS Files ( S3 ), and Queues ( SQS ) and more

ORM

Light-weight ORM in a Domain-Driven Design style

APIs

Build "Universal APIs" that can run on the Web, CLI or from a Queue
